#Firefox OS app#


##Installation##

###1.  Clone this repository###

###2.  Install volo###
	npm install -g volo

You may need to use `sudo npm install -g volo` if your user doesn't have admin permissions.

###3.  Run the app!###
	cd intelliTest/app
	volo serve

###4.  Visit in browser or Firefox OS Simulator###
	http://localhost:8008
